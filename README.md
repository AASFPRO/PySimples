# PySimples

O **PySimples** é aplicativo destinado para crianças de 8 a 12 anos, no intuito de introduzi-las a esse mundo da Programação, que em muitas vezes é tratado como uma coisa extremamente complexa, até para adultos.

A aparência do aplicativo foi projetada para ser o mais agradável possível.
Estruturado em níveis com uma progressão bem gradual, fazendo com que tenha um reconhecimento inicial sobre a linguagem Python.

Em cada nível, Destaca-se algumas atividades que podem ser feitas fora do aplicativo.
